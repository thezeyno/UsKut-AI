# 🧠 UsKut AI

## Kurum İçi / Offline Yapay Zeka Asistanı

UsKut AI, veri güvenliği ve KVKK odaklı geliştirilen, tamamen lokal ortamda çalışabilen bir yapay zeka asistanı projesidir.

Proje; internet bağlantısına ihtiyaç duymadan kullanıcıya sohbet, PDF soru-cevap, veri analizi ve kod düzeltme gibi özellikler sunmayı amaçlar.

---

# 🇹🇷 Türkçe

## 📌 Proje Hakkında

Günümüzde birçok yapay zeka sistemi bulut tabanlı çalıştığı için kullanıcı verileri dış sunuculara gönderilmektedir. Bu durum özellikle kurum içi belgeler ve hassas veriler açısından güvenlik riskleri oluşturabilir.

UsKut AI, bu probleme çözüm üretmek amacıyla geliştirilmiştir.

Sistem tamamen lokal çalışır ve kullanıcı verileri cihaz dışına çıkmadan işlenebilir.

# 🇬🇧 English

## 📌 About The Project

UsKut AI is an offline artificial intelligence assistant developed with a focus on data privacy and institutional security.

Unlike cloud-based AI systems, UsKut processes data locally without sending user information to external servers.

The project aims to provide a secure and controllable AI environment for institutions and individual users.

---

## 🚀 Features

### 💬 AI Chat System

* Real-time chat
* Local LLM integration
* Fast response mode
* Privacy-focused structure

### 📄 PDF Question & Answer

* PDF upload support
* Context-based answering
* Document analysis
* Reduced hallucination behavior

### 📊 Excel / CSV Analysis

* CSV and Excel file processing
* Dataset analysis
* Missing value detection
* Basic data visualization

### 🛠️ Code Fixer

* Buggy code analysis
* Code improvement suggestions
* Debugging assistance

---

## 🧰 Technologies Used

* Python
* Streamlit
* Ollama
* Pandas
* PyPDF
* SQLite

---

## 🔒 Security Approach

UsKut AI is designed to:

* Process data locally
* Reduce external server dependency
* Improve institutional data security
* Provide a KVKK/privacy-oriented structure

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/thezeyno/UsKut.git
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run an Ollama model

Example:

```bash
ollama run llama3
```

### 4. Start the Streamlit application

```bash
streamlit run app.py
```
