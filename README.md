# UsKut-AI
## Türkçe
**📌 Proje Hakkında** 
UsKut AI, kurum içi kullanım için geliştirilmiş, internet bağlantısına ihtiyaç duymadan çalışabilen, gizlilik (KVKK) odaklı çok amaçlı bir yapay zeka asistanıdır. Yerel büyük dil modelleri (Ollama) ile çalışır ve verileri dışarıya göndermez.

**🚀 Özellikler**
💬 Genel sohbet (Chat)
📄 PDF Soru–Cevap (sadece PDF içeriğine dayalı)
📊 Excel okuma, filtreleme ve grafiklerle görselleştirme
🧩 Kod analizi ve düzeltme (Code Fixer)
🌍 Türkçe / İngilizce arayüz
🗂️ SQLite tabanlı olay ve hata loglama
⏱️ Oturum ve idle (zaman aşımı) yönetimi

**🛠️ Kullanılan Teknolojiler**
Python
Streamlit
Ollama (Local LLM)
SQLite
Pandas / NumPy
PyPDF

**🎯 Amaç**
Kurum içinde güvenli, kontrol edilebilir ve genişletilebilir bir yapay zeka asistanı altyapısı sunmak.

## English
**📌 About the Project**
UsKut AI is a privacy-first, offline AI assistant developed for internal and enterprise use. It runs without internet access, uses local large language models (Ollama), and ensures that data never leaves the local environment.

**🚀 Features**
💬 General chat
📄 PDF Question & Answer (based only on document content)
📊 Excel reading, filtering, and visualization with charts
🧩 Code analysis and bug fixing
🌍 Turkish / English interface
🗂️ SQLite-based event and error logging
⏱️ Session and idle timeout management

**🛠️ Tech Stack**
Python
Streamlit
Ollama (Local LLM)
SQLite
Pandas / NumPy
PyPDF

**🎯 Goal**
To provide a secure, offline, and extensible AI assistant infrastructure for internal organizational use.
